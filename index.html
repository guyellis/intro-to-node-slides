  • Create an empty directory (e.g. "statuscheck")
  • From command/console, run:
    • npm install request
  • in this directory.
  • While this is running...
Edit/create file called index.js in this directory and open it in an editor. e.g.:
subl index.js
Add code:
var request = require('request');
request('http://linksilk.com', function(error,response,body){
console.log(response.statusCode);
});
Replace http://linksilk.com with one of your or your client’s websites.
Save file.
(No error if request got installed.)
At terminal run “node index.js”, expect error:
module.js:340
throw err;
^
Error: Cannot find module 'request'
npm install request
node index.js, expect HTTP status such as 200, 301, 404 etc.
Edit console.log and put site name in front:
console.log('linksilk.com: ' + response.statusCode);
Run: node index.js
Expect: Site + status
Duplicate 3 request lines and change second set of lines to a different URL. Use www.linksilk.com
Expect 200 for second result and not 301.
Why did this happen and not 301?

var options = {
